## 🎓 كيفية الصيانة والتطوير

### إضافة منتج جديد:

```json
// في products.json
{
  "ID": "3",
  "name": "Product Name",
  "description": "Short description",
  "image": "images/product.jpg",
  "fullDescription": "Detailed description..."
}
```

### إضافة ترجمة جديدة:

```json
// في ar.json و en.json
{
  "new_key": "translated_text"
}

// في HTML
<element data-i18n="new_key">English text</element>
```

### تخصيص الألوان:

```css
/* في style.css */
:root {
  --primary: #0a7cff;
  --secondary: #00b894;
  /* ... */
}
```
